## Option 2: Copy-Paste (No link yet?)

If iCloud link isn't working yet:

1. Open **Shortcuts** app
2. Tap **+** (top right)
3. Tap **"Add Action"**
4. Search and add these 3 actions in order:

   **Action 1:** Search "URL" → Paste: 
   ```
   https://gustavoibenc.github.io/intermezzos/intermezzos.json
   ```

   **Action 2:** Search "Dictionary" → Add "Get Dictionary Value"
   - Key: type `delivered[0].fact`

   **Action 3:** Search "Show" → Add "Show Result"

5. Tap **⋯** (three dots) at top
6. Name it: `Intermezzo Fact`
7. Tap **Share icon** (square with arrow)
8. Scroll down → **"Add to Home Screen"**
9. Tap **Add** (top right)

**Done!** Widget appears on home screen ✨

**Total time:** 60 seconds

## ❓ Troubleshooting

**"Shortcut not found"?**
- Make sure you're on iPhone/iPad
- Have iOS 14 or later
- Shortcuts app must be installed (free from App Store)

**Widget not updating?**
- Pull down on home screen to refresh
- Or open Shortcuts app once

**Wrong fact showing?**
- Widget shows random fact each refresh
- That's normal!
