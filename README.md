### Method 2: Copy-Paste Setup (3 min)

1. Open **Shortcuts** app on iPhone
2. Tap **+** → "Add Action"
3. Search: **"URL"** → Add it
4. Paste: `https://gustavoibenc.github.io/intermezzos/intermezzos.json`
5. Search: **"Get Dictionary Value"** → Add it
   - Key: `delivered[0].fact`
6. Search: **"Show Result"** → Add it
7. Tap **⋯** → Name it: `Intermezzo Fact`
8. Share → **"Add to Home Screen"**
9. Done!
