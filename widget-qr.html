🍎 Get Intermezzo Widget

Add a daily facts widget to your iPhone home screen in under 1 minute!

⚡ Method 1: Super Quick (30 sec)

  1. Open Shortcuts app
  2. Tap + (top right)
  3. Search "URL" → Add action
  4. Paste this:
    https://gustavoibenc.github.io/intermezzos/intermezzos.json
  5. Search "Dictionary" → Add "Get Dictionary Value"
  6. Set Key to: delivered[0].fact
  7. Search "Show" → Add "Show Result"
  8. Tap ⋯ → Name: Intermezzo Fact
  9. Share icon → Add to Home Screen
💡 TIP: After adding, long press the widget to customize icon color!
